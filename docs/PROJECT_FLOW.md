# Digital File Movement & Approval System

## Project Workflow

Employee/User
    ↓
Login/Register
    ↓
Create File Request
    ↓
Department Assignment
    ↓
Officer/Admin Review
    ↓
Approve / Reject / Return
    ↓
Remarks Added
    ↓
Status Updated
    ↓
Employee Tracks File Progress

---

## Main Modules

1. Authentication Module

4. Dashboard Module
5. Search & Filter Module
